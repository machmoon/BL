# Photos Directory

This directory is for screenshots and photos of the BentleyLibrary application.

## Recommended Screenshots

1. **Home Page** - Main landing page
2. **Search Results** - Search functionality in action
3. **Book Details** - Individual book page
4. **Checkout Form** - Book checkout interface
5. **Checkin Form** - Book checkin interface
6. **Admin Interface** - Django admin panel
7. **Advanced Search** - Advanced search features

## File Naming Convention

- `screenshot-home.png`
- `screenshot-search.png`
- `screenshot-book-details.png`
- `screenshot-checkout.png`
- `screenshot-checkin.png`
- `screenshot-admin.png`
- `screenshot-advanced-search.png`

## Image Specifications

- **Format**: PNG or JPG
- **Recommended Size**: 1920x1080 or similar
- **File Size**: Keep under 2MB per image
