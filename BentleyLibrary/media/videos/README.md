# Videos Directory

This directory is for demo videos of the BentleyLibrary application.

## Recommended Demo Videos

1. **Full Application Walkthrough** - Complete tour of all features
2. **Checkout Process** - Step-by-step checkout demonstration
3. **Search Functionality** - Advanced search features demo
4. **Admin Features** - Admin interface and CSV export demo

## File Naming Convention

- `demo-full-walkthrough.mp4`
- `demo-checkout-process.mp4`
- `demo-search-features.mp4`
- `demo-admin-features.mp4`

## Video Specifications

- **Format**: MP4 (H.264 codec recommended)
- **Resolution**: 1920x1080 (1080p) or 1280x720 (720p)
- **Duration**: 2-5 minutes for full walkthrough, 30-60 seconds for feature demos
- **File Size**: Keep under 50MB per video if possible

## Recording Tips

- Use screen recording software (OBS, QuickTime, Loom, etc.)
- Show clear navigation and interactions
- Include voiceover or captions explaining features
- Start with a brief overview of what you'll demonstrate
