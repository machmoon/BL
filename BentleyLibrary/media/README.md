# Media Files

This directory contains media files for the BentleyLibrary project.

## Structure

- `photos/`: Screenshots and photos of the application
- `videos/`: Demo videos showing application functionality

## Usage

Add screenshots and demo videos here to showcase the application.

## File Naming Convention

- Photos: `screenshot-[feature]-[date].png`
- Videos: `demo-[feature]-[date].mp4`

Example:
- `photos/screenshot-search-2024.png`
- `videos/demo-checkout-2024.mp4`
